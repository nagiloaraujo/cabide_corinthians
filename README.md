# Cabide Corinthians

Landing page promocional para apresentar o cabide premium personalizado do Corinthians, com foco em impacto visual, experiência de produto e captação de leads.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand

## Scripts

```bash
npm install
npm run dev
npm run check
npm run build
```

## Estrutura

- `src/pages/Home.tsx`: composição da landing page
- `src/components/sections`: seções principais da experiência
- `src/components/hero/ProductShowcase.tsx`: vitrine interativa do produto
- `src/components/forms/WaitlistForm.tsx`: formulário de lista de espera
- `src/app/theme.ts`: temas e acabamentos do produto

## Observações

- Os assets principais usados na apresentação estão em `public/`
- O formulário está preparado para prototipação local, salvando leads no `localStorage`
